Frequently Asked Questions

1. What is the WeeeCore Bot?

The WeeeCore Bot is an educational robot kit designed to teach students IoT, AI, and robotics through fun, hands-on activities.

2. What features does it support?

It supports voice coding, IoT integration, rich sensor arrays, line-following mode, and app control.

No prior programming is required. Kids can start with drag-and-drop coding and progress to advanced IoT/AI applications.
