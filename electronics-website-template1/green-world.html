    Python Education Kit - ESP32 Robot Kit

    Details

    Weeemake Our Green World – Python Learning Kit is an affordable entry-level Python education hardware set. It is consisting of an ESP32 mainboard, open-sources electronic modules, durable metal structure parts, Scratch and Python programming software, and teaching tutorials. Let students step into Python programming, understand the meaning and importance of energy conservation and emission reduction, and can learn the Python language-based robot programming.

    Python Education Kit - ESP32 Robot Kit
    ELF ESP32 Mainboard - Python Robot

    ELF ESP32 Controller
    ELF ESP32 is an open-source main control board that is developed for learning Python. There are 2.4G main cores, 1 main button, 1 power switch, 1 USB slot, 1 DC port, and 8 digital ports.

    ELF ESP32 perfectly supports graphical programming software (WeeeCode), so programming materials are very easy for drag-and-drop blocks, support online debugging and offline control. Support both graphical programming such as “Theory Kit, IDLE Shell Editor”, and so on.

    Why WeeeCode?

    • Graphical programming is a reliable and easy tool for beginners to learn programming and can also make it faster and easier for experienced programmers to develop projects.
    • WeeeCode uses drag-and-drop graphical programming that makes learning to code fun and simple.
    • It supports Python programming so users can progress from graphical coding to text-based coding.
    • WeeeCode supports online debugging and offline program control, flexible for practical applications with both fun and fast debugging.
    • Suitable for STEM education, competitions, and project-based learning activities.
